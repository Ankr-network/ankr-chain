# How to install ankrchain

# Install from released version

You can download the latest released version [ankr-chain-v1.0.0]

# Install from source codes

# Install go

Download the stable version go from https://golang.org/dl/, and then install it according to the installing instructions.

# Install ankrchain

  1. clone the ankr-chain repo into you local pc: git clone https://github.com/Ankr-network/ankr-chain.git;

  2. Build ankr-chain image in the ankr-chain root dir:

    make (please make sure that your os has been installed make tool)
