# How to run ankrchain

# Run a local node

# Configurate your node running machine

The basic running machine meets with the following requirements:

4GB Memory, 50GB Idle Disk, OS:Linux/Mac/Windows 10 above

# Run

run the image according to the following steps:

  • Initialize the node configuration: ./ankrchain init

  • Start node:
    ./ankrchain node

# Run in a private network

You can operate according to the following steps:

1.Configurate four node running machines.Each of them meets with the following requirements:

4GB Memory, 100GB Idle Disk, OS:Linux/Mac/Windows 10 above

2.Based on the above four initial files genesis.json under the config dir, generate a new genesis.json for our node such as：
